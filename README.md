# Student Management System in Django

This is a Django-based web application to manage student records efficiently.

---

## 🚀 Features
- Add Student
- Update Student
- Delete Student
- View Student List

---

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/krutika2308/Student-Management-System-in-Django.git

2. Navigate to project folder:
   cd itp_project

3. Run the server:
   python manage.py runserver
