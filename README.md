# Institute Management System (Django)

This is a Django-based web application designed for managing an institute. Users can view available courses, register themselves, and log in to access student-related data.

---

## 🚀 Features
- User Registration & Login Authentication
- View Available Courses
- Access Student List (after login)
- Add, Update, Delete Student Records
- Secure access using Django Authentication

---

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS
- SQLite (default database)

---

## ▶️ How to Run

1. Clone the repository  
2. Navigate to project folder:
   cd itp_project  
3. Run the server:
   python manage.py runserver
